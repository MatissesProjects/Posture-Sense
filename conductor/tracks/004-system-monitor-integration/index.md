# Track 004: System Integration

This track focuses on the "Improving" part: understanding the user's workspace geometry (monitors) and providing suggestions for ideal text placement.

## Status: *Not Started*

## Goals
- Detect Windows monitor layout (primary vs. secondary, stacked geometry).
- Suggest/Determine the optimal pixel coordinate for active text/windows.
- Contextual awareness (differentiating between typing, reading, and reaching).
