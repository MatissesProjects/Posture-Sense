# Track 001: Core CV Engine

This track focuses on the foundational computer vision work: capturing the webcam feed and using MediaPipe to detect key joints.

## Status: *Completed*

## Goals
- Stable webcam capture.
- Real-time MediaPipe Pose integration.
- Eye/Iris tracking integration for gaze direction.
- Export joint coordinates to a standard format (JSON).
