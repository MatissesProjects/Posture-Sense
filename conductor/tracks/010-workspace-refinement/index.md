# Track 010: Workspace Geometry Refinement

## Objective
Fix the workspace model to support asymmetrical monitor layouts (as seen in `monitors_and_webcam_layout.png`) and provide a UI for manual refinement.

## Scope
- Multi-axis monitor offset support.
- Webcam anchoring logic (attaching webcam to a specific monitor edge).
- UI Workspace Editor (drag/nudge monitors).
- Persistent configuration storage.
