# Track 003: Web Interface

This track is the "telling" part: the visual and auditory feedback for the user. It includes the FastAPI backend and a React/Next.js frontend.

## Status: *Not Started*

## Goals
- React-based real-time dashboard.
- FastAPI server with WebSockets.
- Visual feedback on posture (color-coded skeleton, "Posture Score" gauge).
- Low-latency data streaming from CV to UI.
